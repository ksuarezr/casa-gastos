📌 Reglas de aportacion

Define el % del salario que destina cada uno a las casas. La app calcula automaticamente las aportaciones cada mes.

👨 Kev: Aportacion total = X% salario - gastos fijos. Asturias = budget completo. Madrid = resto.
👩 Juls: Aportacion = X% salario. Cubre Madrid restante + gastos fijos. Si no alcanza, se resta del ahorro.
